## 🎯 功能详细说明

### 1. 基础功能 (Core Features)

#### 图像加载
- **描述**：从指定目录加载图像文件
- **支持格式**：JPG, PNG, BMP, GIF
- **实现状态**：✅ 完成
- **测试状态**：✅ 通过

#### 图像浏览
- **描述**：浏览加载的图像，支持上一张/下一张
- **功能**：顺序浏览、循环浏览
- **实现状态**：✅ 完成
- **测试状态**：✅ 通过

#### 标注创建
- **描述**：通过拖拽创建边界框标注
- **支持类别**：watermark, logo, text
- **实现状态**：✅ 完成
- **测试状态**：✅ 通过

#### 标注编辑
- **描述**：编辑已创建的标注框
- **功能**：调整位置、大小、类别
- **实现状态**：✅ 完成
- **测试状态**：✅ 通过

#### 标注删除
- **描述**：删除选中的标注框
- **方式**：右键菜单、键盘快捷键
- **实现状态**：✅ 完成
- **测试状态**：✅ 通过

#### 标注保存
- **描述**：保存标注数据到文件
- **格式**：YOLO 格式 (.txt)
- **实现状态**：✅ 完成
- **测试状态**：✅ 通过
